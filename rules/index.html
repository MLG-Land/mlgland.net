Rules

As Minecraft is rated for children under the age of 13, you will not be allowed to use any vulgar language in the server. Since the Discord ToS state that you must be 13 or older you are free to use any non-racist language in the discord server.

2. No Spamming

You are not allowed to spam in both the Minecraft and Discord servers. This includes using commands in a short period of time. This rule is for your own convenience. (The server could get overloaded if you spam too much)

3. No Advertising

You are not allowed to advertise in the server. This includes using the server to advertise for other servers.

4. No Cheating/Hacking

You are not allowed to use any hacks or cheats in the server. This includes using known minecraft exploits to your advantage.
